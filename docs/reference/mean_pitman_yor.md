# Mean of the n. of species from a Pitman-Yor process

Mean of the n. of species from a Pitman-Yor process

## Usage

``` r
mean_pitman_yor(alpha, sigma, n)
```

## Arguments

- alpha:

  Precision parameter of the Pitman-Yor process

- sigma:

  Strength parameter of the Pitman-Yor process

- n:

  Number of units
